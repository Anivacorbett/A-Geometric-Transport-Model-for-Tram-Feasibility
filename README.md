# A Geometric Transport Model for Tram Feasibility

This project presents a Python-based simulation model analysing how urban geometry and tram placement influence transport accessibility.

The model compares tram and walking travel times across siplified geometric city grids representing different urban forms, including rectangular, triangular, and circular geometries.

For each origin–destination pair, the model determines whether travelling via tram is faster than walking, allowing the percentage accessibility benefit of the tram network to be measured across the city.

Developed as part of a final year university project.
